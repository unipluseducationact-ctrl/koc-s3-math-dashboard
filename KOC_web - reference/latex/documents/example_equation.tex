\section*{Area of a Triangle}
Given base $b$ and height $h$,
\[
  A = \frac{1}{2} b h.
\]

\section*{Pythagoras}
For a right triangle with legs $a$, $b$ and hypotenuse $c$,
\[
  a^2 + b^2 = c^2.
\]
