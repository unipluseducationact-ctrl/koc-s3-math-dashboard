Problem Statement

In the figure, $ABCD$ is a parallelogram. Let $E$ be a point lying on $AD$ such that $AE : ED = 2 : 5$. $CB$ is produced to the point $F$ such that $BF = DE$. Denote the point of intersection of $AB$ and $EF$ by $G$. It is given that $BD$ and $CG$ intersect at the point $H$. Parallelogram Area Problem

Part 1 — Ratios & △AEG ~ △BFG

1

Mark the segment lengths

$AE:ED = 2:5$, so write $AE = 2k$ and $ED = 5k$. Since $BF = DE$ (given), $BF = 5k$ too. And $CB = AD = 7k$ (opposite sides of $ABCD$):

$$DE = 5k, \quad EA = 2k, \quad CB = 7k, \quad BF = 5k$$
2

Prove △AEG ~ △BFG (AAA)

Mark $AD \mathrel{//} CF$ (parallel chevrons). Then:

$$\angle AEG = \angle BFG \;\; (\text{alt. } \angle\text{s, } AE \mathrel{//} BF)$$ $$\angle EAG = \angle FBG \;\; (\text{alt. } \angle\text{s, } AE \mathrel{//} BF)$$ $$\angle AGE = \angle BGF \;\; (\text{vert. opp. } \angle\text{s})$$ $$\therefore \triangle AEG \sim \triangle BFG \;\; (\text{AAA})$$
3

Area of △BFG = 300

Extract the two triangles side by side: $AE = 2k$ against $BF = 5k$. With $[\triangle AEG] = 48$:

$$\left(\frac{AE}{BF}\right)^2 = \frac{[\triangle AEG]}{[\triangle BFG]}$$ $$\left(\frac{2}{5}\right)^2 = \frac{48}{[\triangle BFG]} \implies [\triangle BFG] = 300$$
4

Area of △CBG = 420

$\triangle GCB$ and $\triangle GBF$ share the same height (altitude from $G$ to line $CF$), so areas scale with the bases $CB : BF = 7 : 5$:

$$\frac{[\triangle CBG]}{[\triangle FBG]} = \frac{CB}{BF} = \frac{7}{5}$$ $$[\triangle CBG] = \frac{7}{5} \times 300 = 420$$

Part 2 — △CHB ~ △CGF → 245 & 175

5

DEFB → DB // EF

$DE = BF$ and $DE \mathrel{//} BF$ (both lie on the parallel pair $AD \mathrel{//} CF$), so $DEFB$ is a parallelogram. Mark the double parallel arrows:

$$\therefore DB \mathrel{//} EF \;\;(\text{so } HB \mathrel{//} GF)$$
6

Prove △CHB ~ △CGF (AAA)

With $HB \mathrel{//} GF$ marked:

$$\angle CHB = \angle CGF \;\; (\text{corr. } \angle\text{s, } HB \mathrel{//} GF)$$ $$\angle CBH = \angle CFG \;\; (\text{corr. } \angle\text{s, } HB \mathrel{//} GF)$$ $$\angle GCB \;\; \text{common}$$ $$\therefore \triangle CHB \sim \triangle CGF \;\; (\text{AAA})$$

Then lift a copy of $\triangle CHB$ (with its marked angles) out of the figure to compare it against $\triangle CGF$ directly.

7

Area of △CHB = 245

$CB = 7k$ and $BF = 5k$, so $CF = 12k$ and $[\triangle CGF] = 420 + 300 = 720$:

$$\left(\frac{CB}{CF}\right)^2 = \frac{[\triangle CHB]}{[\triangle CGF]}$$ $$\left(\frac{7}{12}\right)^2 = \frac{[\triangle CHB]}{720} \implies [\triangle CHB] = 245$$
8

Area of △GBH = 175

Lower $\triangle CHB$ back onto $\triangle GCB$ — the leftover red region is $\triangle GBH$:

$$[\triangle GBH] = 420 - 245 = 175$$

Part 3 — △CDH ~ △GBH → 343

9

Prove △CDH ~ △GBH (AAA)

$ABCD$ is a parallelogram, so $CD \mathrel{//} AB$, i.e. $CD \mathrel{//} GB$ (double parallel marks). Then:

$$\angle CDH = \angle GBH \;\; (\text{alt. } \angle\text{s, } CD \mathrel{//} GB)$$ $$\angle DCH = \angle BGH \;\; (\text{alt. } \angle\text{s, } CD \mathrel{//} GB)$$ $$\angle DHC = \angle BHG \;\; (\text{vert. opp. } \angle\text{s})$$ $$\therefore \triangle CDH \sim \triangle GBH \;\; (\text{AAA})$$
10

Find GB and CD in k

From $\triangle AEG \sim \triangle BFG$ (ratio $2:5$, step 2): $AG : GB = AE : BF = 2 : 5$, so $AG = 2k$, $GB = 5k$. And $CD = AB = AG + GB = 7k$:

$$AG = 2k, \quad GB = 5k, \quad CD = 7k$$
11

Final answer

$$\left(\frac{GB}{CD}\right)^2 = \frac{[\triangle GBH]}{[\triangle CDH]}$$ $$\left(\frac{5}{7}\right)^2 = \frac{175}{[\triangle CDH]}$$
$$[\triangle CDH] = 343 \text{ cm}^2$$

1

Set up the figure

$ABCD$ is a parallelogram. Top line $DEA$ and bottom line $CBF$ are parallel (single arrows on each). Mark $G = AB \cap EF$ and $H = DB \cap CG$ for later steps.

2

Opposite parallel bases

Highlight $EA$ (top) and $BF$ (bottom). These are the opposite bases of the bowtie pair:

$$EA \parallel BF \quad (\text{both on } AD \parallel CF)$$
3

Bowtie centre $G$

Lines $AB$ and $EF$ cross at $G$. Each triangle uses one opposite base and two segments through $G$ — the classic bowtie at $G$.

4

Draw △EAG and △GBF

Shade $\triangle EAG$ (top) and $\triangle GBF$ (bottom). They share vertex $G$ only; bases lie on opposite parallels.

5

Prove △EAG ~ △GBF (AAA)

$$\angle AEG = \angle BFG \;\; (\text{alt. } \angle\text{s})$$

$$\angle AGE = \angle BGF \;\; (\text{vert. opp. at } G)$$

$$\therefore \triangle EAG \sim \triangle GBF \;\; (\text{AAA})$$
